# 🌱 Carbon Track

A modern web platform to **monitor, analyze, and reduce carbon emissions** across mines in India.
Carbon Track enables organizations to upload data, generate insights, and track sustainability performance through an interactive dashboard and leaderboard.

---

## 🚀 Features

### 🏭 Mine Management

* Add and manage multiple mines
* Store location, subsidiary, and operational details

### 📄 Document Upload & Analysis

* Upload reports and documents related to emissions
* Automated data extraction and processing

### 📊 Smart Dashboard

* Visualize carbon emissions data
* Track:

  * Total emissions
  * Quarterly trends
  * Reduction progress
* Identify largest emission sources

### 💡 Actionable Insights

* AI-powered recommendations using Gemini API
* Highlights:

  * High-priority actions
  * Quick wins for emission reduction
  * Strategic sustainability suggestions

### 🏆 Leaderboard

* Ranks mines across India based on:

  * Carbon reduction progress
* Encourages competition and accountability

---

## 🧠 Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS

### Backend

* Node.js (Express)
* Hosted on Render

### Database

* Supabase (PostgreSQL)

### AI Integration

* Gemini API (for insights & analysis)

### Deployment

* Frontend: Vercel
* Backend: Render

---

## ⚙️ How It Works

1. Add a mine
2. Upload emission-related documents
3. System processes and stores data
4. Dashboard displays insights and trends
5. AI generates actionable recommendations
6. Mines are ranked on leaderboard

## 📦 Installation (Local Setup)

```bash
# Clone repo
git clone https://github.com/your-username/carbon-track.git

# Install frontend
cd client
npm install

# Install backend
cd ../server
npm install
```

---

## 🔐 Environment Variables

### Frontend (.env)

```
VITE_API_URL=your_backend_url
```

### Backend (.env)

```
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_key
GEMINI_API_KEY=your_gemini_key
```
